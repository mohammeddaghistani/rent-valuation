# نظام دعم قرار التقييم الإيجاري (إصدار جاهز للاستخدام الداخلي)

هذا المستودع يحتوي على نظام داخلي متكامل:
- تسجيل دخول + أدوار (Admin / Committee / Valuer / DataEntry)
- إدارة الصفقات (CRUD) مع ربطها بالموقع (إحداثيات)
- تقييم إيجاري مبدئي يعتمد على الصفقات القريبة + نشاط مماثل + قواعد احتياطية
- درجة ثقة رقمية + وصفية
- رفع كروكي (PDF/صورة) وتخزينه وربطه بالتقييم
- توليد تقرير PDF رسمي للتقييم
- لوحة مؤشرات استراتيجية داخلية (للاستخدام الداخلي)

## النشر على Streamlit Cloud
1) ارفع الملفات (مفكوكة) إلى GitHub (Private).
2) Streamlit Cloud → New app → اختر repo → main → app.py → Deploy
3) ضع أسرار الدخول في Secrets:
- ADMIN_USERNAME / ADMIN_PASSWORD / COOKIE_SECRET

> ملاحظة: استبدل الشعار والخط داخل assets حسب هوية شركتك.
